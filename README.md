# Predictive Analytics Using Historical Data

This project uses Machine Learning and historical sales data to forecast future sales trends and analyze advertising impact using Power BI.

## Project Objectives
- Build a predictive model using historical datasets
- Forecast future sales trends
- Analyze advertising influence on sales
- Evaluate prediction accuracy
- Visualize predictions using Power BI

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Power BI

## Machine Learning Technique
- Linear Regression

## Dashboard Features
- Actual vs Predicted Sales
- Sales Trend Analysis
- Advertising Impact Visualization
- KPI Metrics
- Forecast Visualization

## Key Insights
- TV advertising strongly impacts sales.
- Predicted sales closely match actual sales.
- Historical trends help forecast future performance.

## Files Included
- Python notebook
- Power BI dashboard
- Dashboard screenshots
- Project report
